# ⚽ Predicción de Fútbol con IA y App Móvil

Este proyecto utiliza **Python** para analizar estadísticas de fútbol en tiempo real y envía las predicciones a una aplicación **Android** a través de **Firebase**.

## 🛠️ Tecnologías utilizadas
* **Python (Pandas/Requests)**: Procesamiento de datos.
* **Firebase Realtime Database**: Puente de datos en la nube.
* **Football-Data.org**: Fuente de estadísticas reales.
* **GitHub**: Control de versiones.

## 📈 Funcionamiento
El script analiza la relación entre puntos obtenidos y goles a favor para determinar la "potencia" actual de cada equipo y predecir un posible ganador.
